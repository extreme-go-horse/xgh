# collaboration-dispatcher

Multi-agent workflow orchestration -- manages collaboration threads, dispatches work items, monitors progress, and routes messages between agents.

## Model

sonnet

## Dispatched by

`collab` command

## Workflow templates

- **plan-review** (2 agents): Planning + review
- **parallel-impl** (N agents): Parallel implementation tasks
- **validation** (2 agents): Cross-validation
- **security-review** (chain): Sequential security audit

## Tools

Read, Grep, Glob
