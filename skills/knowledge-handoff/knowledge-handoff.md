# xgh Knowledge Handoff

Use this skill when ownership transfers between developers or agents.

## Handoff Checklist

1. Summarize current scope, constraints, and known risks.
2. Include links to relevant context-tree entries and manifests.
3. Record outstanding TODOs and validation gaps.
4. Curate the handoff note so follow-up sessions can continue without rediscovery.

A good handoff minimizes restart time and confusion.
