# xgh Onboarding Accelerator

Use this skill to bootstrap new contributors quickly.

## Onboarding Flow

1. Load top context-tree entries by maturity and impact.
2. Summarize architecture map, workflows, and conventions.
3. Provide first-task recommendations with low risk.
4. Highlight common pitfalls and where to find verification commands.

Strong onboarding helps contributors become productive in fewer sessions.
